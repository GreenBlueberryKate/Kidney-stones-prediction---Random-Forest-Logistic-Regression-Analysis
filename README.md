# Kidney-stones-prediction---Random-Forest-Logistic-Regression-Analysis

Kidney Stone Prediction using Machine Learning

This project explores whether kidney stones can be predicted from urine analysis measurements using interpretable machine learning models in Python.

Dataset

The dataset contains laboratory urine measurements such as:

specific gravity (gravity)

pH level (ph)

osmolality (osmo)

conductivity (cond)

urea concentration (urea)

calcium concentration (calc)

Target variable:

0 — no kidney stone

1 — kidney stone present

Dataset source: Kaggle — Kidney Stone Prediction Based on Urine Analysis.
